About the SmartPost Project

This "Smart Post" represents a culmination of our engineering studies, fusing **Machine Learning** with practical web application development to solve a real-world marketing problem. The project demonstrates not only technical proficiency but also an understanding of data-driven business intelligence.

The core innovation is the **predictive model**, which is trained to estimate **Impressions**—a key measure of audience reach—based on early engagement signals (Likes, Comments, Shares, Saves) and the content format (Reel, Video, etc.). This allows creators to iterate on content strategy *before* a post gains widespread visibility.

Project Architecture & Technologies:

  • Backend (API): Python Flask, offering a robust and simple REST API layer.
  • Machine Learning: Utilizing a proxy regression model (e.g., Scikit-learn, persisted via `joblib`) for key metric prediction.
  • Frontend (UX/UI): Modern Single Page Application (SPA) using HTML5, vanilla JavaScript for logic, and Tailwind CSS for a highly responsive, glass-morphic design.
